# AutoMétho_Windows
  LISEZ CE TEXTE AVANT DE TÉLÉCHARGER LE PROGRAMME
  
  Un programme pour traiter des liens durant des projets de recherche et les formatter d'une façon convenable. Ayant reçu le lien url, le programme cherche automatiquement les informations pertinentes (l'auteur, le titre...) et les mettent dans un format approprié. Destiné à être un projet de bénévolat pour les élèves de mon école.
  
  Si tu utilises une machine MacOS ou Linux, bonne chance. Il est fortement possible que le programme ne peut pas fonctionner hors de Windows, mais je ne sais pas. Vous êtes bienvenus de l'essayer et même de m'aider à créer une version qui fonctionnera pour les utilisateurs de MacOS.
  
  Il faut qu'on soit très clairs ici: AutoMétho n'est qu'un outil pour se faciliter la tâche et enlever le besoin de traiter des dizaines de liens url à la main. Son propriétaire n'est en aucun cas responsable d'erreurs dans les résultats offertes par le programme. Une vérification manuelle sérieusement et rigoureusement effectuée est essentielle à une utilisation saine de cet outil. L'outil atteint les standards attendues et proclamées par son propriétaire seulement si cette vérification est effectuée et que ce texte a été lue avant tout contact avec cet outil. Toute tentative de forcer le propriétaire à apporter un changement ou un mise à jour sur l'outil obtiendra invariablement une réponse négative. Cependant, le propriétaire s'engage à être attentif aux problèmes des utilisateurs et à y rémédier autant qu'il juge possible.

  En ce qui concerne comment télécharger cet outil, il est très simple. Sur cette page, dans le coin haut-droite, trouvez un bouton vert qui dit "Code". Cliquez dessus, vous devriez être dans la moitié "local". En bas, cliquez sur "download zip". Il faut ensuite extraire le dossier du fichier zip et la placer à un endroit souhaitable. Sortez le dossier AutoMétho V1.X Windows. Utilisez WinRAR sur le fichier "ms-playwright.rar" et choisissez "extraire ici". Celui-ci devrait ressembler à l'image. Si ceci est le cas, vous êtes prêts à utiliser le programme. Consultez les Figures 1 au besoin.

  En ce qui concerne comment utiliser cet outil, il est très simple. En premier, dans le fichier "Input links.txt", mettez vos liens bruts. Il devrait déjà y avoir une panoplie de liens divers pour indiquer comment les placer. Si vous essayez de tester si le programme fonctionne, n'y touchez rien. Les liens déjà présentes servent justement à tester le programme. Quand vous êtes prêts, double-cliquez sur le fichier "main.exe". Deux fenêtres devraient apparaître: un noir, un blanc. Le fenêtre noir ne sert strictement que de faire peur à l'utilisateur, donc n'y prêtez pas attention. Dans le fenêtre blanc, cliquez sur le bouton "Commencer le programme?" et, comme indiqué, patientez. Le processus de chercher des informations en ligne prend malheureusement quelques secondes. Quand le bouton dit "Tâche Complétée. Quitter le programme?", cliquez dessus. Les deux fenêtres devraient se fermer. Pour récupérer les sources formattées, ouvrez le fichier "output.docx" et simplement prenez les résultats qui sont là. Consultez les Figures 2 au besoin. 

  Dans le cas d'un erreur majeur et d'une incapacité totale de traiter un lien, veuillez consulter le fichier "program logs.txt" avant de contacter le propriétaire de cette programme. Si du texte coloré apparaît dans le fenêtre noir, il est fort probablement important aussi. Si vous contactez le propriétaire de cette programme, donnez-lui ces informations aussi.

  À date, cette programme se limite dans la détection, le triage et le traitement des ouvrages de référence en ligne, des périodiques en ligne et des sites web. Faites attention: tout ce qui n'est pas identifié comme un ouvrage de référence en ligne ou un périodique en ligne est considéré automatiquement un site web. Il est donc possible que des types de sources plus rares, tel un rapport officiel ou une base de données soit identifié comme un site web ou blogue alors qu'il ne l'est pas. Ceci est une des raisons de l'importance de revérifier à la main. En plus, l'outil prévoit agrandir son champ de compétence vers les audiovisuels.
