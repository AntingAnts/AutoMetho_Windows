# AutoMétho_Windows
  LISEZ CE TEXTE AVANT DE TÉLÉCHARGER LE PROGRAMME
  
  Un programme pour traiter des liens durant des projets de recherche et les formatter d'une façon convenable. Ayant reçu le lien url, le programme cherche automatiquement les informations pertinentes (l'auteur, le titre...) et les mettent dans un format approprié. Destiné à être un projet de bénévolat pour les élèves de mon école.
  
  Si tu utilises une machine MacOS ou Linux, bonne chance. Il est fortement possible que le programme ne peut pas fonctionner hors de Windows, mais je ne sais pas. Vous êtes bienvenus de l'essayer et même de m'aider à créer une version qui fonctionnera pour les utilisateurs de MacOS.

  En ce qui concerne comment télécharger cet outil, il est très simple. Sur cette page, dans le coin haut-droite, trouvez un bouton vert qui dit "Code". Cliquez dessus, vous devriez être dans la moitié "local". En bas, cliquez sur "download zip". Il faut ensuite extraire le dossier du fichier zip et la placer à un endroit souhaitable. Sortez le dossier AutoMétho V1.X Windows. Celui-ci devrait ressembler à l'image. Si ceci est le cas, vous êtes prêts à utiliser le programme <img width="750" height="300" alt="image" src="https://github.com/user-attachments/assets/64414913-fffe-49f2-b049-6c28ac576c16" />

  En ce qui concerne comment utiliser cet outil, il est très simple. En premier, dans le fichier "Input links.txt", mettez vos liens bruts. Il devrait déjà y avoir une panoplie de liens divers pour indiquer comment les placer. Si vous essayez de tester si le programme fonctionne, n'y touchez rien. Les liens déjà présentes servent justement à tester le programme. Quand vous êtes prêts, double-cliquez sur le fichier "main.exe". Deux fenêtres devraient apparaître: un noir, un blanc. Le fenêtre noir ne sert strictement que de faire peur à l'utilisateur, donc n'y prêtez pas attention. Dans le fenêtre blanc, cliquez sur le bouton "Commencer le programme?" et, comme indiqué, patientez. Le processus de chercher des informations en ligne prend malheureusement quelques secondes. Quand le bouton dit "Tâche Complétée. Quitter le programme?", cliquez dessus. Les deux fenêtres devraient se fermer.
