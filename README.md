# AutoMetho_Windows
Un programme pour traiter des liens durant des projets de recherche et les formatter d'une façon convenable. Ayant reçu le lien url, le programme cherche automatiquement les informations pertinentes (l'auteur, le titre...) et les mettent dans un format approprié. Destiné à être un projet de bénévolat pour les élèves de mon école.
